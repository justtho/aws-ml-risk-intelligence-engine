# AWS Integration Layer

This folder contains AWS integration utilities for the AWS ML Risk Intelligence Engine.

## Planned AWS Components

- Amazon S3 for dataset, logs, and model artifact storage
- Amazon Bedrock for AI-generated risk summaries
- Amazon SageMaker for future model training and deployment
- Amazon OpenSearch for future semantic vector search
- Amazon CloudWatch for logging and monitoring

## Current Status

Local MVP is complete. AWS integration layer is being added incrementally.
